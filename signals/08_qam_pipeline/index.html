QAM 16/64 전체 복조 파이프라인

Gray 코딩 → I/Q 변조 → AWGN 채널 → 코히런트 복조 → AGC → 성상도 → BER

복조 파이프라인
1
비트열 → Gray 코딩 → I/Q 레벨 매핑 → 변조 신호 s(t)
2
채널: AWGN 노이즈 + 위상 오프셋 + 진폭 감쇠 → r(t)
3
r(t) × 2cos(2πfc·t+φ̂) → I, r(t) × −2sin → Q (코히런트 복조)
4
적분+덤프 (Integrate & Dump) → 심볼당 I/Q 샘플
5
AGC 진폭 보정 → 최소 거리 판정 → 비트 복원 → BER
