Fourier Transform — core equation
F(ω) = ∫₋∞^∞ f(t) · e^{−jωt} dt   ← Forward transform
f(t) = (1/2π) ∫₋∞^∞ F(ω) · e^{jωt} dω   ← Inverse transform
The 3D visualizer (S01) shows: viewing along Z → time domain f(t) | viewing along X → frequency spectrum |F(ω)|. Each basis function e^{−jωt} traces a helix in 3D space.
